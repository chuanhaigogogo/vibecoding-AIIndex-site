# 🚀 AI Index 

> 一个精选的 AI 核心资源信息索引，拒绝废话，直达干货。

> 制作初衷：每天打开各种app追ai消息，有点手忙脚乱。有时会忘看某部分，有时觉得退出进入不同app很烦。所以罗列了日常主要看的消息源头，自动跳转，当check list一个个看完。

---

### 🌟 项目亮点
* **精选源**：只收录真正有价值的AI信息源。
* **自带 Key 模式**：支持 BYOK (Bring Your Own Key)，保护隐私，零成本调用。
* **极简设计**：没法再简了。

### 🛠️ 如何使用总结功能？
1. 点击网页上的“总结”按钮。
2. 在弹窗中输入你的 **DeepSeek API Key**（或其它兼容接口）。也可以使用别的ai api，自行改改。
3. 你的 Key 将存储在浏览器本地（LocalStorage），本站后台不会记录。
